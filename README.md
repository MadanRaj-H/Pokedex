# Pokedex
Awesome Pokedex mobile App

The app lists all different types of POKEMON which includes Pokemon's
  * Attack
  * Defense
  * Height
  * Weight
  * Description
  * Next level of Evolution

  2) pod init

  3) Go to https://github.com/Alamofire/Alamofire for installing Alamofire pod init.
